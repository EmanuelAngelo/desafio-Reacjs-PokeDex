## Detalhes do projeto
Consumir a API e listar todos os pokémons que a consulta do seguinte endpoint retornar:
https://pokeapi.co/api/v2/pokemon

**Você deve exibir NOME, EXPERIÊNCIA e IMAGEM de cada pokémon.**
